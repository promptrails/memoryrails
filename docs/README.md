# MemoryRails

> Agent memory management for Go LLM applications.

Store, embed, and retrieve memories using pluggable embedding providers and vector stores. Supports 5 memory types, importance scoring with decay, and semantic similarity search.

## Install

```bash
go get github.com/promptrails/memoryrails
```
